# Scientific Calculator in C

A menu-driven scientific calculator built using the C programming language.

## Features

- Basic Arithmetic Operations
  - Addition
  - Subtraction
  - Multiplication
  - Division

- Scientific Functions
  - Square
  - Square Root
  - Power (a^b)
  - Factorial
  - Exponential (e^x)
  - Natural Logarithm (ln)
  - Logarithm (log10)

- Trigonometric Functions
  - sin
  - cos
  - tan
  - sec
  - cosec
  - cot

- Temperature Conversion
  - Fahrenheit to Celsius

- Matrix Operations
  - Matrix Addition
  - Matrix Subtraction
  - Matrix Multiplication

## Language

- C
